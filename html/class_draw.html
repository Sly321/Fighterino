Detailed Description

Die Draw Klasse (ruft die Klassen Character, Background und UiOverlay auf).

Die Klasse Draw ruft die Klassen Character, Background und UiOverlay auf. Diese sind für das Zeichnen im Spiel verantwortlich. Character und Background werden Zahlen übergeben welche für die jeweilige Auswahl stehen (Background 1, 2, ... etc.). Die UiOverlay Klasse brauch zum Zeichnen immer den Character als object, damit Healthbar etc. korrekte Werte zeichnen.
