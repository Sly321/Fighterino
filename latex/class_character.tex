\subsection{Detailed Description}
\hyperlink{class_character}{Character} Klasse ist für die Darstellung und Bewegungen der Charaktere zuständig. 

Diese Klasse ist vollständig für das Zeichnen und Bewegen des Charakters zuständig. Sie wird von \hyperlink{draw_8cpp_source}{Draw.\+cpp} aufgerufen. Um neue Bilder einzufügen muss nur im Contructer ein Case hinzugefügt werden, z.\+B. Case 2. Nun kann dann im Auswahlmenu für Charaktere die 2 gewählt werden (Mitte oben) und die Bilder dieses Charakteres werden geladen. Achte darauf alle Bilder, und auch auf die richtige größe eingefügt zu haben. Sonst wird es nicht funktionieren. Shadow kann so geladen werden wie er ist, er wird für alle Charaktere benutzt.
